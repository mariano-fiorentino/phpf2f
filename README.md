# phpf2f
Name
----

phpf2f - (php) (4) (to) php(5)

Description
-----------

Phpf2f (php4 to 5) is a shell script that migrate php4 code into php5 syntax.


Usage: phpf2f --phpcli=/usr/bin --phpini=/etc --workdir=/var/www/htdocs [--tryonly] [--verbose] [--logfile=/path/to/log]
